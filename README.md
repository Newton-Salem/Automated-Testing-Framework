# 🧪 Automated Testing Framework

This is a QA Automation Testing Framework built to cover different types of software testing including:

- UI Automation Testing
- API Testing
- Manual Testing
- Performance Testing
- Database Testing
- CI/CD Integration

---

## 📌 Project Structure
Automated-Testing-Framework/

- api-tests/  
  Postman collections & environments  

- automation/  
  Selenium Java framework (POM design pattern)  

- ci-cd/  
  Jenkins pipelines and CI configuration  

- database/  
  SQL scripts and database validation  

- docs/  
  Test plans and documentation  

- manual-tests/  
  Manual test cases  

- performance/  
  JMeter performance test plans  

- README.md

## 🛠️ Tools & Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven
- Postman
- Apache JMeter
- Jenkins 
- Jira 
- Git & GitHub 

---

## 🎯 Project Objectives

- Build a scalable automation framework
- Cover different testing levels (UI, API, Performance)
- Apply real-world QA practices
- Integrate CI/CD pipeline for continuous testing
- Improve software quality and reliability

## 📊 Performance Testing

Performance tests are created using Apache JMeter:

- Location: `performance/jmeter/test-plan.jmx`
- Open using Apache JMeter and run the test plan
